## 🚀 Featured Projects

### 🚲 [Bike Data Lakehouse](https://github.com/emmanayoola/databricks-bike-lakehouse_2026)
> End-to-end Data Lakehouse built with Databricks and the Medallion Architecture

- Ingested structured data from 6 ERP & CRM CSV sources into a Delta Lake Bronze layer
- Cleaned, validated and standardised data through a Silver transformation layer (6 notebooks)
- Designed a Star Schema Gold layer: `fact_sales`, `dim_customers`, `dim_products`, `dim_date`
- Automated the full pipeline with Databricks Jobs and orchestration notebooks

**Stack:** `Databricks` `PySpark` `Python` `SQL` `Delta Lake` `Medallion Architecture`

---

### 💰 [Financial Data Pipeline](https://github.com/emmanayoola)

- Integrated financial data via REST APIs with missing-value handling and normalisation
- Applied feature engineering to deliver AI-ready, well-documented datasets
